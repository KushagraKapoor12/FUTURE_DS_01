# Task 1: Business Sales Performance Analytics

## Objective
Analyze business sales data to identify high-performing categories, sub-categories, and regional sales distribution.

## Dataset
Sample Superstore Dataset

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboard

## Key Insights
- Technology is the highest revenue-generating category.
- Phones and Chairs are the top-selling sub-categories.
- The West and East regions contribute the most to overall sales.
- Office Supplies generate steady sales across regions.

## Dashboard
A consolidated dashboard was created to visualize category-wise sales, top sub-categories, and regional performance.

## Note
The dataset does not contain date fields, so trend analysis over time was not performed.
